# NX tRPC v10 Test

Most basic implementation of:

- [NX](https://nx.dev/)
- [tRPC v10](https://trpc.io/)
- [Prisma](https://www.prisma.io/)

## Development

```bash
git clone git@github.com:nowlena/nx-trpc-test.git
cd nx-trpc-test
yarn
yarn start
```

## Directory Structure

```
.
├── apps
│   ├── frontend              # NextJS App
│   ├── frontend-e2e          # NextJS App E2E Tests
├── libs
│   ├── api                   # tRPC
│   ├── prisma                # Prisma
```
